## Overview

The **Coffee Sales Analysis Dashboard** is a visually engaging and data-driven business intelligence solution that transforms raw coffee sales data into clear, actionable insights. Designed for managers and decision-makers, this dashboard delivers a complete picture of sales performance — from daily trends to top-selling products — all in one interactive view.

It highlights key revenue drivers, uncovers customer buying patterns, and identifies high-impact opportunities such as peak sales hours, best-performing stores, and product-wise profitability. With intuitive visuals, smart filters, and real-time analytics, the dashboard empowers stakeholders to make faster, smarter, and more profitable business decisions.


## Features

- Includes interactive controls such as Month Filter and Day Filter for flexible analysis

- Features essential KPIs: Total Sales, Total Footfall, Average Bill per Person, and Average Orders per Person

- Provides detailed visual insights through Hourly Quantity Trend Chart

- Offers category-level understanding with Category-Wise Sales Pie Chart and Order Size Distribution Pie Chart

- Compares performance across locations using the Store-Wise Footfall & Sales Chart
